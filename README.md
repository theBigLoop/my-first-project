Домашнее задание по Скай на Питоне номер 1

Техническое задание :

1. Программа здоровается и предлагает начать:
Привет! Предлагаю проверить свои знания английского!
Напиши, как тебя зовут. - ГОТОВО

2. Программа считывает имя пользователя и говорит:
Привет, имя_пользователя, начинаем тренировку! - ГОТОВО

3. Программа задает по очереди три вопроса
Если ответ правильный, приложение говорит:
Ответ верный!
Вы получаете 10 баллов!
Если ответ неверный, говорит:
Неправильно.
Правильный ответ: ______
Затем приложение задает следующий вопрос.  - ГОТОВО

4. После ответа на 3 вопроса приложение говорит:
Вот и всё, имя_пользователя!
Вы ответили на ___ вопросов из 3 верно.
Вы заработали ___ баллов.
Это ____ процентов. - ГОТОВО

Критерии проверки домашнего задания
Программа работает так, как описано в задании.
Переменные назвали корректно.
Код умеренно прокомментировали.
Программа использует f-строки.
